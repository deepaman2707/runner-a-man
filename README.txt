SPACE RUNNER PRO PWA

How to use:
1. Keep the folder structure exactly as provided.
2. Upload the files to GitHub Pages or host on HTTPS.
3. Open the site in Safari on iPhone.
4. Enter your name and upload your photo.
5. Tap Start Game, then use JUMP / SHOOT buttons.
6. To install: Safari > Share > Add to Home Screen

Controls:
- Jump button = Jump
- Shoot button = Shoot
- Pause button = Pause
- Music button = Toggle music

Desktop controls:
- Space / Arrow Up = Jump
- F = Shoot
- P = Pause
- R = Restart after game over

Important:
- If old version still loads, clear Safari website data.
- Delete old Home Screen app and install again after updates.
- Service worker cache version is v4 in this build.
